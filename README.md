# Next.js Project Setup and Basics

## Introduction

This repository is for the initial setup of a Next.js project with TypeScript and Tailwind CSS.
It serves as the foundation for building a modern web application with reusable components, routing, API integration, and responsive design.

## Scaffold the project with TypeScript and Tailwind CSS

## Objective

Scaffold a Next.js project with TypeScript, ESLint, and Tailwind CSS, and set up a basic structure.

## Setup

1. Create the project:

```bash
npx create-next-app@latest alx-project-2
```

Enable TypeScript, ESLint, and Tailwind CSS during setup.

2. Project structure:

```bash
/components/layout/Header.tsx
/interfaces/index.ts
/public/assets/images
```

3. Add a welcome message in `pages/index.tsx`.

4. Run the app:

```bash
npm run dev -- -p 3000
```

Visit [http://localhost:3000](http://localhost:3000) to verify.
